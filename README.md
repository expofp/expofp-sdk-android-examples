# ExpoFP SDK Android Examples

Sample apps demonstrating [ExpoFP SDK](https://expofp.github.io/expofp-fplan-android/) integration for Android.

## Samples

| Sample | Description |
|--------|-------------|
| [mini-map](mini-map/) | Exhibitor list with an interactive mini floor plan that expands full-screen with route directions |

## Requirements

- Android Studio Ladybug or later
- Min SDK 26
- [ExpoFP SDK](https://expofp.github.io/expofp-fplan-android/)

## Getting Started

```bash
git clone https://github.com/expofp/expofp-sdk-android-examples.git
```

Open the project in Android Studio, select the desired sample run configuration, and run on a device or emulator.
